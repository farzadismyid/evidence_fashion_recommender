# Evidence-Constrained Multi-Modal Fashion Recommendation

MPhil Artificial Intelligence research project focused on building a fashion recommendation system that combines:

- Multi-modal understanding (image + text)
- Retrieval-Augmented Generation (RAG)
- Evidence-aware recommendation
- Faithful explainable AI

## Research Goal

Given:

- a fashion item image
- a natural language request

The system recommends complementary fashion items and generates explanations grounded in retrieved fashion knowledge.

## Planned Components

1. Fashion Understanding Layer
2. Fashion Knowledge Base
3. Candidate Retrieval
4. Evidence-Aware Re-Ranking
5. Faithful Explanation Generation
6. Interactive Feedback Loop
